# Node.js Express을 활용한 게시판 프로그램
## 사용Skills: 
####         1. Node.js , Express
####         2. View : BootStrap, Semantic 
####         3. 데이터베이스 :MongoDB
####         4. 기타: 페이스북 API

## 구현기능: 
####         1. 회원가입을통한 로그인 or 페이스북으로 로그인해야만 사용이가능하다(https://localhost:3000) 
####         2. 사용자는 커뮤니티 게시판에 글을 CRUD할수있다.
